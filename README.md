# YouNavi Prompt Templates

[YouNavi](https://younavi.me/) 是一款 AI 驱动的会议分析与决策辅助工具，帮助专业人士从会议记录和日常工作中提取深度洞察，做出更明智的决策。

本仓库是 YouNavi 提示词模板的开源集合，涵盖会议分析、投资决策、创业管理、咨询服务、产品设计、战略规划等多个专业场景。

## 模板分类

| 分类 | 模板数 | 说明 |
|------|--------|------|
| [会议分析](templates/01-会议分析.md) | 10 | 多维度深度理解每一场会议 |
| [启发决策](templates/02-启发决策.md) | 5 | 突破思维定式，做出更明智的决策 |
| [早期投资](templates/03-早期投资.md) | 6 | 覆盖从项目初筛到投后管理的完整流程 |
| [创业公司](templates/04-创业公司.md) | 4 | 融资、方向、团队和节奏全方位支持 |
| [咨询顾问](templates/05-咨询顾问.md) | 3 | 从问题诊断到方案落地的全流程 |
| [产品负责人](templates/06-产品负责人.md) | 4 | 从用户洞察到产品设计的关键环节 |
| [战略负责人](templates/07-战略负责人.md) | 4 | 从机会发现到战略拆解的完整链路 |

## 使用方式

1. 浏览上方分类表，找到你感兴趣的场景
2. 点击链接查看该分类下的所有提示词模板
3. 选择合适的模板，在 YouNavi 中使用
